# Screenshots Directory

Add your dashboard screenshots here after running the application.

## Required Screenshots:

1. **overview.png** - Dashboard overview showing KPIs and main interface
2. **map.png** - Interactive map view of MDPP suppliers
3. **data-table.png** - Data explorer table view

## How to Take Screenshots:

1. Run the app locally: `streamlit run app.py`
2. Navigate through each tab
3. Take screenshots (use Cmd+Shift+4 on Mac or Snipping Tool on Windows)
4. Save them in this directory with the names above
5. Commit and push to update the README images

The README.md file references these screenshots to showcase the dashboard functionality.
